FAQ

Pertanyaan yang sering ditanyakan

Bagaimana cara upgrade plan?
Klik tombol "Beli via WhatsApp" pada plan yang diinginkan. Admin akan memproses upgrade dan memperbarui plan akun kamu dalam waktu singkat.
Apakah limit direset setiap hari?
Ya, limit request direset otomatis setiap hari pada pukul 00:00 WIB. Kamu tidak perlu melakukan apa pun secara manual.
Apa yang terjadi jika limit habis?
Jika limit harian habis, API akan mengembalikan error 429 (Too Many Requests). Limit akan direset otomatis keesokan harinya, atau kamu bisa upgrade plan untuk limit lebih tinggi.
Apakah API key aman?
Setiap API key bersifat unik dan ter-hash. Jika kamu merasa key bocor, kamu bisa regenerate kapan saja dari dashboard. Key lama akan langsung tidak berlaku.
